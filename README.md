
# Syntrax-VitaImage
Syntrax VitaImage is a DISM gui based tool to intergrate drivers, updates and packages to iso's from Windows 7 and newer.

**Requirements**
- .NET 4.7.2
- Windows 7 or higher (The system you are making it on)
- Windows 7 or higher ISO (The ISO you are trying to modify)

**How does it work?**
First, I would recommend flashing an ISO to a usb stick. That way you are saving yourself some headaches. When you did that, Create a folder on the C drive for example "Temp". This is the folder you are gonna use as "Mount folder"

When selecting your WIM file, Navigate to the sources folder in the ISO and first select "boot.wim". Dont forget to intergrate your updates and drivers in **BOTH** .WIM files.
